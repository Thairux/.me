This update marks the start of a more intentional portfolio: cleaner presentation, stronger highlights, and a better way to show both what I build and how I think about the work.

## What's new

- A sharper homepage with recent highlights
- Better project cards that feel more visual and easier to scan
- A portfolio voice that focuses on work, writing, and direction instead of template filler
- Light and dark presentation that both feel polished

## Stack snapshot

The current setup keeps things lightweight and practical:

- Jekyll for content structure
- GitHub Pages for publishing
- Custom CSS for branding, layout, and motion
- Small JavaScript touches for theme control and interaction polish

## What this portfolio should do

I want this site to act like a useful front door: a place where someone can quickly understand what I work on, what I'm interested in, and how I approach building thoughtful software.

## What's next

Next, I want to add stronger project case studies, more writing, and clearer proof of what I'm shipping so the portfolio keeps becoming more personal and more useful over time.
