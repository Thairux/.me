# Portfolio

A minimalist portfolio website built with Jekyll.

## About

This is a personal portfolio website showcasing my projects, skills, and experience. It's built with Jekyll for simplicity and performance.

## Getting Started

### Prerequisites

- Ruby 2.7+
- Jekyll 4.0+
- Bundler

### Installation

```bash
# Clone the repository
git clone https://github.com/Thairux/.me.git
cd .me

# Install dependencies
bundle install

# Run the local server
bundle exec jekyll serve
```

The site will be available at `http://localhost:4000`

## Structure

```
.
├── _posts/           # Blog posts
├── _projects/        # Project showcase
├── _data/            # Site data (YAML)
├── _layouts/         # HTML layouts
├── _includes/        # Reusable components
├── assets/           # CSS, JS, images
├── _config.yml       # Jekyll configuration
└── index.md          # Homepage
```

## Configuration

Edit `_config.yml` to customize:
- Site title and description
- Author information
- Social links
- Theme settings

## Building

```bash
# Build for production
bundle exec jekyll build

# Output goes to _site/ directory
```

## Deployment

Deploy the `_site/` directory to GitHub Pages or any static hosting service.

For GitHub Pages:
1. Push to `main` branch
2. GitHub Pages will automatically build and deploy
